# Chat Application

A real-time chat application built with Node.js, Express, MongoDB, Redis, and Socket.IO.

## Tech Stack
- Backend: Node.js, Express
- Database: MongoDB, Redis
- Real-time: Socket.IO
- Auth: JWT, bcryptjs
